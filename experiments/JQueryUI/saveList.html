Description

This experiment demonstrates the local cache feature of HTML5 using Jquery. HTML5 allows to store objects or collection of objects in a session like variable with is maintained in the browser cache.The local storage object is used to store retrieve objects. In this experiment, we take the name of person and id and on click of save button create and object of this information and store it in local storage under a key value. Every time a new person is added , the values are appended to the existing list of persons. Please check the next experiment for Onload demo.
