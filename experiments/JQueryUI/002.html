Code Snippet

          $(function () {

        $("#bounce").hover(fadeit);

    });

    function fadeit(event)

    {

        var target = $(event.target);

        $(target).toggle("fade");

    }

Description

This experiment the fading effect in Jquery on hovering on an element. The jquery function grabs hold of the hover function for the div and calling a javascript function which toggles between fade in and fade out.
