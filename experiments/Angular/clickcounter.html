Description

The above experiment demonstrates the use ng-click, ng-show and ng-hide. A counter is implemented using ng-click. The value of the counter increments on every click and the value is displayed on UI.ng-show and ng-hide are used to show/hide tags.The show/hide model bound with the checkbox.On checking the checkbox the images are shown / hidden

Code snippet

           < input type="checkbox" ng-model="show" />Show Button
           < img src="image1.jpg" height="250" width="250" ng-show="show" />
