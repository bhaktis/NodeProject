Delete records

About the experiment

In this experiments we will be deleting record to the mongoDB database. The developer objects JSON objects are deleted from the database using the remove() function of the MOngoDB.

Description

The experiment demonstrates deleting new records to the MongoDB collection using the remove() function provided by mongoDB.remove() function is a asyn call that deletes data from the developer collection. Inside this function call we define another functions which will be called on completion of remove with the error data if any and the resultset with the record deleted. This new resulset is send to the node controller which updates the view.

Code snippet

Deleting records

    app.delete("/developers/:id", function (req, res) {
    DeveloperModel.findById(req.params.id, function (err, doc) {
        doc.remove();
        DeveloperModel.find(function (err, data) {
            res.json(data);
        });        
    });   
});
