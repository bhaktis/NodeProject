Add records

About the experiment

In this experiments we will be adding record to the mongoDB database. The developer objects JSON objects are inserted in the database using the save() function of the MOngoDB.

Description

The experiment demonstrates adding new records to the MongoDB collection using the save function provided by mongoDB.Save function is a asyn call that adds the new data to the developer collection. Inside this function call we define another functions which will be called on completion of save with the error data if any and the resultset after adding the new record. This new resulset is send to the node controller which updates the view.
