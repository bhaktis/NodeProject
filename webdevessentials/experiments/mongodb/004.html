Display records

About the experiment

In this experiments we will be fetch the data from mongoDB and show it to the user. This is done using find function of MOngoDB.

Description

The experiment demonstrates retrieving records to the MongoDB collection using the 'find' and 'findById' function provided by mongoDB.find function returns all the records from the developer collection. Inside this function call we define another functions which will be called on completion of find with the error data if any and the resultset. This new resulset is send to the node controller which updates the view.

Code snippet

Displaying records

    app.get("/developers", function (req, res) {
    DeveloperModel.find(function (err, data) {
        res.json(data);
    });  
});
