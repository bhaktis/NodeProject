About the experiment

In this experiments we will be adding record to the mongoDB database. The developer objects JSON objects are inserted in the database using the update() function of the MOngoDB.

Description

The experiment demonstrates update records to the MongoDB collection using the update function provided by mongoDB.Update function is a asyn call that updates data in the developer collection. The update function take some identifier for the record to be updated and the new values . Inside this function call we define another functions which will be called on completion of update with the error data if any and the resultset after updating the record. This new resulset is send to the node controller which updates the view.
