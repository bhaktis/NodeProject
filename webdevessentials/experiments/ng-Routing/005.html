Multiple views in a single page

About the experiment

The routing functionality added by this step is provided by angular in the ngRoute module, which is distributed separately from the core Angular framework. The below experiment we see how to add multiple views to a Single page application.The views will be displayed under the same ng-view.

{{hello}}

Description

Angular js has feature called RouteProvider using which you can add multiple views in an HTML page. For this, we have to configure the config of the mainApp and provide the template URl and the controller for click of the link.We also need to specify the default URL where the page will be directed incase there are no matched found. The page will be viewed in the div for which we have specified the ng-view directive.

Code snippet

                                                 app.config(['$routeProvider', function ($routeProvider) {
    $routeProvider.
        when('/', {
            templateUrl: 'multiple.html',
            controller: 'RouteCtrl'
        }).       
    otherwise({
        redirectTo: 'view/Books/book.html'
    });
}]);
app.controller('RouteCtrl', function ($scope) {
    $scope.template = {
        "books": "views/books/book.html",
        "food": "views/food/food.html"
    }
    $scope.message = {
        "books": " Here is the list of books",
        "food": "List of food items"
    }
    $scope.books = [{ name: "book1", publisher: "publisher1" },
        { name: "book2", publisher: "publisher2" },
        { name: "book3", publisher: "publisher3" },
        { name: "book4", publisher: "publisher4" }
    ];
    $scope.food = [{ name: "item1", date: "date1" },
        { name: "item2", date: "date2" },
        { name: "item3", date: "date3" },
        { name: "item4", date: "date4" }
    ];
});
