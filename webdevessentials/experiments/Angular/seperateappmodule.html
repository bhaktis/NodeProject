About the experiment

Enter last name:

Name: {{student.fullname()}}

Description

The above experiment demonstrates validation for input fields using Angular.The example maintains the angular js application in a seperate file.This is done by declaring the ng-app field for the div and calling the controller of the module using this field.The $dirty and $error states are used to validate data and track the errors.These states are shown using the ng-show field in the span tag. The validation are provided for empty fields.

Code snippet

        < input name="firstname" type="text" ng-model="student.firstname" required />
        < span style="color:red" ng-show="form.firstname.$dirty && form.firstname.$invalid">
        < span ng-show="form.firstname.$error.required"> Name is required
