Resizing a responsive page

About the experiment

This is a responsive page which will scale according to the brower size. It will scale for web / tablet as well as mobile browsers.

Description

This page demonstrates using HTML and CSS3 for creating responsive web application. resize the window and this page will resize itself to fit the window.

Code snippet

using Viewport 

                    < meta name="viewport" content="width=device-width, initial-scale=1" />
