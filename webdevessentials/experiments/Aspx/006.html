Responsive page using media query

About the experiment

The @media rule is used to define different style rules for different media types/devices. In CSS2 this was called media types, while in CSS3 it is called media queries. Media queries look at the capability of the device, and can be used to check many things, such as: width and height of the browser window width and height of the device orientation (is the tablet/phone in landscape or portrait mode?) resolution

Description

This page has width set if the page is viewed on a small screen the content is rendered accordingly. Shrink the page size to see how it works !!

This is a responsive page done with media query

Modal main Div
