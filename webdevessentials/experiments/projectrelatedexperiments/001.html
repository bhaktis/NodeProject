Understanding Amazon and ebay Product API's

About the experiment

The web application user can shop for products using Amazon and eBay API's. The user can shop from amazon and ebay. The product information is retrieved from the Amazon and eBay API's.

Description

Amazon API's : Prerequisites for consuming the amazon API: Need to install amazon API key, shared key and associate key. For this, we need to create an affiliate account with amazon. For consuming amazon api's we need to install the amazon-aws package which makes available the amazon API's for node.Using this client we create an instance of the product advertisement client providing the amazon API key, shared key and the associate key. Ebay API's : Prerequisities : Create an affiliate account with eBay. Install the node rest client and make a rest API call to the ebay service with the API key as parameter.The web service returns data in JSON format.
