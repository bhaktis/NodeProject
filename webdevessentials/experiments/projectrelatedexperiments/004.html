Rendering the amazon content using angular JS

About the experiment

The data from amazon api rendered in the UI using angular JS.

Description

Amazon API's : Prerequisites for consuming the amazon API: Need to install amazon API key, shared key and associate key. For this, we need to create an affiliate account with amazon. For consuming amazon api's we need to install the amazon-aws package which makes available the amazon API's for node.Using this client we create an instance of the product advertisement client providing the amazon API key, shared key and the associate key. The UI rendered angular js.
