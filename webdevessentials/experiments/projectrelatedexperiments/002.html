Listing Amazon Products on web page.

About the experiment

The web application user can shop for products using Amazon and eBay API's. The product information is retrieved from the Amazon API's.

Description

Amazon API's : Prerequisites for consuming the amazon API: Need to install amazon API key, shared key and associate key. For this, we need to create an affiliate account with amazon. For consuming amazon api's we need to install the amazon-aws package which makes available the amazon API's for node.Using this client we create an instance of the product advertisement client providing the amazon API key, shared key and the associate key.
