CRUD operations using Angular, Node Js and MongoDB

This is an implementation of a basic CRUD operations using Angular,Node and MongoDB.

In the 1st experiment we will set-up the connection string used to connect MongoDb with Node js and setting up Open Shift for the same.

2nd, 3rd, 4th and 5th experiments will Insert, delete, select and update operations performed on data stored in MongoDB

Following operation are performed on the data:
1. Get data - Using the http get method - Select Command
2. Add field - using post method of HTTP - Save Command
3. Edit fields - Edit table fields using the put method of http - Update Command
4. Delete data - Using the delete method - Remove command
